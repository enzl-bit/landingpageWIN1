OptimizeWin10

Tool PowerShell aman & modern untuk mempercepat Windows 10, hapus bloatware, dan restore fitur penting.

Download Sekarang

Fitur Utama

  • Hapus bloatware yang aman tanpa merusak sistem.
  • Optimasi performa sistem dan layanan latar belakang.
  • Restore fitur penting Windows yang sering dinonaktifkan otomatis.
  • Profil pengguna (Lite, Balanced, Advanced) dengan keterangan masing-masing.
  • Undo/Restore pengaturan sistem sebelum optimasi.
  • Auto update dengan log perubahan (changelog).
  • Tampilan GUI modern berbasis Windows Forms.
  • Script ditandatangani digital & open source di GitHub.

Cara Menggunakan

  1. Download ZIP dari tombol di atas atau kunjungi GitHub.
  2. Ekstrak isi file ke folder mana saja.
  3. Jalankan file OptimizeWin10.ps1 dengan klik kanan → "Run with PowerShell".
  4. Pilih profil optimasi sesuai kebutuhan Anda.
  5. Ikuti panduan GUI untuk mengaktifkan atau menonaktifkan fitur.
  6. Gunakan fitur restore jika ingin membatalkan perubahan.
